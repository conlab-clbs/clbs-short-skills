# 冒頭フック分類軸（解剖用）

> 出典: ショート動画講義資料 + 常識破壊パターン集（株式会社コンサルタントラボラトリー）
> リサーチで異常値の冒頭2秒を解剖するとき、この分類で「型」を判定する。
> ここは**分類のための辞書**。台本執筆の作法は `clbs-short-script` 側に置く。

---

## A. 冒頭2秒フック 5型（必ずどれかに分類する）

| # | 型 | 心理効果 | 見分け方 |
|---|---|---|---|
| ① | **常識破壊（意外性）** | 認知不協和 | 視聴者の常識を一文で覆す。「実は逆」「〇〇は幻想」 |
| ② | **ネガティブ訴求** | プロスペクト理論（損失回避） | 失敗・痛み・危険の予告。「〇〇な人の特徴3選」「超危険」 |
| ③ | **短期的快楽** | 楽・簡単・早い | 即時メリット。「1日1分で」「たった3000円で」 |
| ④ | **質問系** | 好奇心の隙間 | 答えが気になる問いかけ。「〇〇って知ってる？」 |
| ⑤ | **コレをやると◯◯** | サイガニック（続きが気になる） | 結論を隠してベネフィット or 地獄を予告。「"コレ"をやると一生〇〇」 |

> 多くの異常値は **テロップ＝①〜⑤のどれか** ＋ **音声＝興味づけ or 本題の入り**、という分担になっている。
> 解剖時は「テロップがどの型か」「音声がどの型か」を別々に判定するとよい。

## B. 常識破壊フレーム 8種（タイトル/テロップの構造分類）

| フレーム | 型 |
|---|---|
| 1. 実は"逆"だった | 「実は○○な人ほど××をしていない」「多くの人がやってる○○、実は逆効果」 |
| 2. 信じられていた○○は幻想 | 「○○すれば安心、は幻想です」「正しいと思ってた○○、時代遅れ」 |
| 3. ○○してるのにうまくいかない | 「努力してるのに成果が出ない理由は…」 |
| 4. その常識いつ作られた？ | 「あなたの"当たり前"、今の時代には通用しません」 |
| 5. ○○は一部の人だけ | 「○○すればうまくいく？実は一部の人だけの話」 |
| 6. 努力が報われない本当の理由 | 「報われないのは戦う場所が違うから」 |
| 7. 意外と知られてない落とし穴 | 「便利そうに見えて○○には落とし穴がある」「知らないと損する○○」 |
| 8. 成果が出る人は○○してない | 「うまくいってる人はむしろ○○をやらない」 |

## C. 「不」起点（ペルソナの痛みを冒頭に置く型）

不安・不満・不足・不快・不信 → これを冒頭で言い当てる（カクテルパーティー効果＝「私のことだ！」）。
変換ルール: ①反対に言い換える ②常識を疑い逆を突く ③ありがちな行動の落とし穴を指摘。

## D. 心理トリガー 8種（切り口の分類）

① 常識破壊 ② ネガティブ訴求 ③ 短期的快楽 ④ カクテルパーティー（当事者意識）
⑤ カリギュラ（禁止・悪用厳禁） ⑥ サイガニック（続きが気になる） ⑦ 希少性・権威性 ⑧ まとめ・網羅・ランキング

## E. 解剖の観点チェックリスト（各異常値で埋める）

冒頭1〜2秒について:
- [ ] **何が写っているか**: 顔アップ / 手元 / 文字のみ / 映像素材 / 人物＋オブジェクト
- [ ] **構図・動き**: ズーム / カット割り / ジェスチャー / 目線
- [ ] **焼き込みテロップ**: 出ている文字を**そのまま**（フォント強調色・改行位置も）
- [ ] **音声の冒頭ひと言**: 聞こえたまま
- [ ] **テロップの型**: A 5型のどれか ＋ B フレーム何番
- [ ] **音声の型**: A 5型のどれか（テロップと役割が違うことが多い）
- [ ] **視覚×言語の分担**: テロップで結論/権威を見せ、音声で問い/煽りを言う等
- [ ] **権威ソースの有無**: 「○○大学の報告によると」「ノーベル賞」等が冒頭に挿入されているか

冒頭以降（簡潔に）:
- [ ] **興味づけ**: 「やらないと最悪な未来／やれば最高の未来 → 保存して何度も見て」型があるか
- [ ] **本題の見せ方**: 図/表/映像素材/会話調 のどれで深さを出しているか
- [ ] **CTA**: コメント誘導 / プロフィール誘導 / フォロー誘導（続編フック）
